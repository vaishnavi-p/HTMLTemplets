Regression Model Evaluation

  1. R2 Score:
    R^2 (coefficient of determination) regression score function. The best possible score can be 1 and it can go negative as well in worst conditions.
  2. Mean squared error:
    The mean_squared_error function computes mean square error, a risk metric corresponding to the expected value of the squared error or loss.
  3. Mean absolute error:
    The mean_absolute_error function computes mean absolute error, a risk metric corresponding to the expected value of the absolute error loss. Here is a code example which demonstrates the usage of various evaluation methods for regression models
